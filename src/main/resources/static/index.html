App Historial Médico

Objetivos

Facilitar el tiempo para las personas, instituciones tanto privadas como estatales para la solicitud y verificación de las historias clínicas de los pacientes que por primera vez son atendidos en un centro de salud tanto privado como estatal. Por ello se creará este Aplicación Web que se conectara a un web services y se podrá consultar los datos de los pacientes.

  • -Aumenta la rapidez en la atención del paciente.
  • - Reduce la demora que existe cuando se necesita consultar la historia clínica de un paciente.
  • -Verificación exacta del estado del paciente(confiabilidad).
  • -Proporcionar la historia clínica de un paciente tanto en los hospitales nacionales como clínicas privadas.
